PROMPT: Return the number of times that the string "hi" appears anywhere in the given string.


countHi("abc hi ho") → 1
countHi("ABChi hi") → 2
countHi("hihi") → 2

CODE: 
public int countHi(String str)
{
  int hiCount = 0;
  for(int i=0;i+1<str.length();i++)
  {
    if(str.charAt(i)=='h'&&str.charAt(i+1)=='i')
      hiCount++;
  }
  return hiCount;
}
